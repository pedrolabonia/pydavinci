    ProjectManager.db property writable

    Gets or sets current database.

    For setting a Disk DB 
    ProjectManager.db = {
'DbType': 'Disk',
'DbName': 'Local Database'
}
    For setting a PostgresSQL db 
    ProjectManager.db = {
'DbType': 'PostgreSQL',
'DbName': 'PostgresDB',
'IpAddress': '127.0.0.1'
}

    Parameters:

    Name Type Description Default
    db_info dict

    valid db_info dict.

    		 required
